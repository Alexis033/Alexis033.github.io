My projects

School enrollment management system

The project aims to develop a document management system for student registration using Bootstrap for the user interface design, React JS for the frontend, Python as the main programming language, and FastAPI as the framework for developing the RESTful API. The goal of the project is to simplify and streamline the student registration process, allowing them to easily and quickly upload the required documents, and providing system administrators with an efficient tool to manage and approve the received documents.

Tic-tac-toe

A tic tac toe game created using React JS and CSS, with options to play against the computer or another player locally. It includes features to save the user's score and the last unfinished game. The main goal is to provide an enjoyable and visually appealing gaming experience, while also allowing users to easily track their progress over time.

TextLock

Text encryptor using HTML, CSS, and JavaScript. The project includes a form that allows the user to enter a text and encrypt or decrypt it. In addition, the web page has additional features such as the ability to copy the encrypted or decrypted text to the user's clipboard, as well as deleting the entered text using a button.

Todo-list

This project is a task list application built with React and TypeScript. It allows users to create, edit, and delete tasks, as well as mark them as completed. The application features a user-friendly interface and uses local storage to persist task data.

Other projects with React
